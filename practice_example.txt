Starting a Fire

1 Survey area
  1.1 Make note of obstacles
2 Choose a fire site
  2.1 Choose a flat area
  2.2 Avoid rocks and trees
3 Prep site
  3.1 Clear away debris
    3.1.1 Pick up loose rocks
    3.1.2 Pick up loose leaves
    3.1.3 Pick up loose sticks
  3.2 Create ring of rocks
    3.2.1 Gather rocks
    3.2.2 Place in circle
    3.2.3 Add additional circle
4 Place kindling
5 Start fire
  5.1 Strike flint
    5.1.1 Hit flint with rock over kindling
    5.1.2 Repeat strike flint until you get a spark
  5.2 Blow on spark until spark grows
  5.3 Add logs
    5.3.1 Place small logs on fire
    5.3.2 Place medium logs on fire
    5.3.3 Place large logs on fire

Writing an Essay

1. Format essay
  1.1 Choose font
    1.1.1 Choose font style
    1.1.2 Choose font size
  1.2 Set margins
  1.3 Set line spacing
2. Draft essay
  2.1 Draft intro
    2.1.1 Create a hook
    2.1.2 Introduce key concepts
    2.1.3 Write a thesis
      2.1.3.1 State your main theme
      2.1.3.2 State your main points
  2.2 Draft body
    2.2.1 Write first paragraph
      2.2.1.1 Write intro sentence
      2.2.1.2 Make main point
      2.2.1.3 Write conclusion sentence
  2.3 Draft conclusion
    2.3.1 Summarize main points
    2.3.2 Remind reader of reflections
3. Edit essay
  3.1 Check for spelling mistakes
  3.2 Read through draft
    3.2.1 Read it out loud to check for flow
    3.2.2 Ensure punctuation is accurate
  3.3 Check to ensure it hits length requirements
